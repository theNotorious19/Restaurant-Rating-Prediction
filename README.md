# 🍽️ Restaurant Rating Prediction

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on a restaurant dataset using Python and Pandas. The objective is to understand restaurant information, clean the dataset, and prepare it for machine learning tasks.

---

## 🚀 Features

- Load restaurant dataset
- Explore dataset structure
- Check missing values
- Data cleaning
- Basic Exploratory Data Analysis (EDA)
- Dataset preprocessing

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## 📂 Project Structure

Restaurant-Rating-Prediction/
│── Dataset.csv
│── task-1.ipynb
│── README.md
│── requirements.txt

---

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/theNotorious19/Restaurant-Rating-Prediction.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Open the notebook

```bash
jupyter notebook task-1.ipynb
```

Restaurant dataset containing restaurant information such as:
